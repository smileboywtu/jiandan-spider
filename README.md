# Jiandan Spider

Scrape jiandan/ooxx images and save to file system.

This is a demo:

![Alt text](./screenshot/demo.png)

# Feature

This program powered by python 2.7.

Support:

1. Auto Proxy
2. HTTP REQUEST Load Setting
3. Multiple Page Scrape

# Requirements

+ lxml
+ gevent

# How-To

Just run the spider.py

``` shell
python spider.py
```
